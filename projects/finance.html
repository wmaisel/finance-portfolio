DCF Valuation App

React + Vite front end with FastAPI and yfinance backend to run discounted cash flow scenarios in a simple UI.

  • Pulls market data via yfinance and normalizes inputs
  • Interactive UI for revenue growth, margins, and discount rate sensitivities
  • API endpoints that encapsulate valuation logic for reuse
Visit Repository Run Locally Instructions

Run Locally

Follow these steps to run the DCF app locally:

  1. git clone <repo-url>
  2. cd backend
  3. pip install -r requirements.txt
  4. uvicorn main:app --reload
  5. cd ../frontend
  6. npm install
  7. npm run dev
